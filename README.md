
# Sai Vivek — Portfolio (Next.js + Tailwind)

A sleek, Apple‑inspired portfolio site built with Next.js (App Router), Tailwind CSS, Framer Motion, and lucide‑react.

## 🧰 Tech
- Next.js 14 (App Router)
- Tailwind CSS 3
- Framer Motion
- lucide‑react icons
- TypeScript

## 🚀 Quickstart
```bash
# 1) Install deps
npm install

# 2) Run dev server
npm run dev

# 3) Open
http://localhost:3000
```
